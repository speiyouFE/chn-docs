    # 监听消息 (on) +2.0.0

    Worker进程统一监听互动消息。

    比如监听来自 websocket, render, main等进程的消息。

    # 基础用法

    // 引入文件
const { getMessageCenter } = require('../message-center')
const messageCenter = getMessageCenter()

// 调取接口
messageCenter.on(command, callback)

    # 示例

    messageCenter.on(ipcCmds.SHOW_STUDENT_BTN, (message) => {
  mLogger.info(`收到弹屏显示学生端按钮message:${JSON.stringify(message)}`)
  if (message.show) {
    this.showChildBtnWin()
  } else {
    this.hideChildBtnWin()
  }
})

    # 参数

    参数 说明 类型 必填 可选值
    command 监听消息的指令。注意：指令必须是一一对应的 String ✔ -
    callback 收到的消息后需要执行的回调 Function ✔ -
