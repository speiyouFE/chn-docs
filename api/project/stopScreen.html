# 隐藏窗口

跳转到空场景并隐藏窗口

// 引入函数
import { stopScreen } from './tools'

// 调用函数
stopScreen()

::: info 调用这个函数，会执行如下动作：

  • 跳转到空场景
  • 隐藏窗口
  • 发送一条IPC指令 :::
